Zarr Explorations

JavaScript examples using zarr-layer for interactive map visualizations

Basic Zarr Example
Demonstrates loading and visualizing a simple local Zarr dataset using MapLibre GL and zarr-layer. This dataset has no extra (non-xy) dimensions to select.
Local Data • zarr-layer • MapLibre GL
Weather Data Example
Visualizes ERA5 weather data from a remote Zarr store, showing rainfall patterns for 2017. Demonstrates time-series data selection and remote data access. This dataset includes a time dimension, and we select a specific year.
Remote Data • ERA5 • Time Series
Public S3 Bucket Example
Demonstrates loading Zarr data from a public S3 bucket. Shows how to work with publicly accessible cloud-hosted Zarr datasets.
Public S3 • Cloud Data
Private S3 Bucket Example (Not Supported)
Private S3 buckets are not currently supported by zarr-layer. This example is preserved for reference but will not work. See this issue for updates.
Private S3 • Not Supported
